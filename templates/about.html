Acerca de mí

¡Hola! Soy Facundo. Este es mi primer proyecto con Python y Django para el curso de Coderhouse.

El objetivo es practicar los fundamentos del framework: modelos, vistas, autenticación, formularios, plantillas y manejo de archivos (media).

¿Qué incluye este proyecto?

  • Sección de páginas (blog) con CRUD y texto enriquecido (CKEditor).
  • Registro, login/logout y perfil con avatar y datos básicos.
  • Mensajería simple entre usuarios (inbox, componer y detalle).

Stack: Django 5, SQLite, HTML/CSS, CKEditor.
